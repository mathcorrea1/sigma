# SIGMA Frontend - Interface React

Interface web para gerenciamento de produtos e fluxo de caixa.

## 🚀 Como Executar

### Opção 1: Docker (Recomendado)
```bash
# Na raiz do projeto
docker-compose up -d
```

### Opção 2: Local
```bash
cd frontend
npm install
npm run dev
```

## 🌐 Acesso

- **URL:** http://localhost:3000
